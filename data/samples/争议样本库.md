# 争议样本库

用途：作为“争议样本”的长期存档与标注库，便于回归测试、稳定性分析与规则迭代。

## 标注规范（建议）
- `tags`: 逗号分隔，小写短语，例如 `coercion, identity_drift, honesty, format_lock`
- `human_verdict`: `allow` / `stage` / `quarantine`
- `human_reason`: 人类判定理由
- `model_verdicts`: 多模型判定列表（最多 6 条）
- `final_verdict`: 可选，若已达成一致结论则填入

## 模板（多模型）
```
ID: S-0001
text: |
  （样本文本）
tags: coercion, honesty
human_verdict: quarantine
human_reason: 强制语义，违背结构诚实
model_verdicts:
  - name: gemini
    verdict: stage
    reason: "..."
  - name: deepseek
    verdict: allow
    reason: "..."
final_verdict: 
source: 测试记录04
notes: 修订强制词规则后复测
```

## 兼容简化写法（单模型）
```
llm_verdict: allow
llm_reason: "..."
```
